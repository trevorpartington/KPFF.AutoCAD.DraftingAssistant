Application.QuitAborted EventDescription
Wraps the AcEditorReactor.quitAborted() ObjectARX function. This notification occurs if a shutdown of AutoCAD is aborted, either by an individual veto() in beginQuit(), or by any other condition that would abort a shutdown such as canceling a prompted save of a document.
