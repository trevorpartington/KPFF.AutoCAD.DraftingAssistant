The X, Y, and Z scaling of the block reference is applied to the entities in the referenced block table record relative to their perceived WCS coordinate axes, which may or may not be the same as the drawingâ€™s WCS axes. 

 

The X scale factor value is used for DXF group code 41. 

 

The Y scale factor value is used for DXF group code 42. 

 

The Z scale factor value is used for DXF group code 43.
